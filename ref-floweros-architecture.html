Ключові потоки даних

👤 Реєстрація

Клієнт: /start → онбординг-анкета
→ Airtable: Users (create)
→ LMachine: create_user(tg_id, referrer_id) WRITE
→ Клієнт отримує loyalty card ID

🔔 Нагадування

Scheduler → Airtable: профілі +2 дні
→ Flora24: get_stock(flowers) READ
→ LMachine: get_balance(user_id) READ
→ Клієнт: "До [дати] 2 дні. Є [квіти]. Кешбек: X грн"

🛒 Замовлення → Оплата

Клієнт: бюджет → обирає спосіб оплати
→ Flora24: create_preorder() WRITE
→ Airtable: Orders (pending_photo)
→ Менеджер: нотифікація → фото → клієнт
Готівка: +3% → менеджер ✅ · Онлайн: webhook ✅
→ LMachine: record_transaction() WRITE

🏆 Loyalty Card

Клієнт: /card або кнопка
→ LMachine: get_user_stats() READ
Бот: баланс · тир · прогрес до наступного тиру
→ LMachine: get_referral_link() READ
Клієнт ділиться посиланням → новий юзер

🔗 Profile Share

Власник: /share → deep link (profile_id)
→ надсилає у будь-якому месенджері
Отримувач: відкриває бот → бачить профіль
→ задає власну мітку (вільний текст)
→ Scheduler нагадує отримувачу про ці дати

🤖 AI Support

[💬 Запитати флориста] → AI Agent
Context: ShopKnowledge + профіль + замовлення
Tools: get_stock() · get_balance() · escalate()
Не впевнений → escalate_to_manager()
→ Менеджер отримує контекст розмови
