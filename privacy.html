Privacy Policy

App: Threads-LeadGen-Artem-org-ua
Last updated: June 12, 2026

1. Overview

This app is a personal automation tool used by Artem Stepanenko to search public Threads content by keyword and engage with relevant posts on behalf of the @artem.org.ua account. It is not a public-facing application and is not distributed to third parties.

2. Data We Access

  • Public Threads posts matching specific keywords, accessed via the Threads API
  • Basic profile information of the authenticated Threads account (user ID, username)
  • Replies to comments posted by the authenticated account

3. How We Use Data

Accessed data is used solely to:

  • Identify relevant public posts to engage with
  • Post comments on public Threads content on behalf of the authenticated user
  • Monitor replies to those comments and facilitate responses

Data is not sold, shared with third parties, or used for advertising.

4. Data Storage

Post IDs and comment logs are stored in a private Google Sheets document accessible only to the app owner. No personal data of third-party Threads users is stored beyond post ID and username as visible on the public Threads platform.

5. Data Retention

Logged data is retained for up to 30 days for deduplication purposes, then no longer referenced.

6. User Rights

This app operates on behalf of a single authenticated account (@artem.org.ua). The account owner has full control over the app and can revoke access at any time via Threads account settings.
