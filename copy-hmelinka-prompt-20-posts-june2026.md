# Промпт для Claude Project @hmelinka — 20 постів

*Вставити в чат проекту Хмелінка на claude.ai*

---

Катерино, ось свіжа аналітика акаунту (50 постів, реальні числа):

**Що зайшло найкраще:**
- Єгипет, басейн, конфлікт із сином → 383K переглядів
- Син — чемпіон світу → 34K, ER 14.65%
- Брат мобілізований → 27K
- Реабілітаційний центр → 4.6K (нещодавній, ще росте)

**Що не зайшло:**
- Абстрактна психологія без особистої сцени → 300–1500 переглядів
- «Самооцінка — це не про...» без конкретного прикладу → 1302 перегляди, 3 лайки

**Висновок:** Особисті мікроісторії з конкретною сценою б'ють теорію в 10–100 разів. Материнство, війна через особистий досвід, несподіваний поворот — це ядро.

---

Напиши 20 готових Threads-постів для черги на наступні 2 тижні (5–6 постів на тиждень).

**Розподіл:**

- **8 постів** — особисті мікроісторії: сини, спорт, побут, несподівані сцени з реального життя (придумуй реалістичні ситуації в дусі реальних постів Катерини)
- **6 постів** — анонімні кейси з практики: конкретна жінка, конкретна ситуація, без імені, з психологічним поворотом наприкінці
- **4 посту** — короткі парадоксальні інсайти через конкретну деталь (не теорія — образ)
- **2 посту** — легалізація «поганих» почуттів (формат: «це нормально...»)

**Для кожного посту:**
1. Текст готовий до публікації (розмовна мова, голос Катерини)
2. У 6 із 20 постів — м'який CTA наприкінці (запис у шапці профілю)
3. Якщо пост довший за 500 символів — розбий на тред: t1 / t2 / t3

**Важливо:**
- Жодного токсичного позитиву
- Жодної позиції зверху
- Конкретна деталь у кожному пості (тілесна або побутова)
- Ритм: короткі речення чергуються з довшими
- «Шо» (не «Що»), «Ну кайф», живі вигуки — якщо підходять по ситуації
- Без em-dash (—), без стрілок (→ ↓)
