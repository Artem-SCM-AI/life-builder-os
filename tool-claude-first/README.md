# Claude Code Перше знайомство

Персональний Claude Code workspace — готовий за 15 хвилин.

---

## Крок 1 — Встанови VS Code

1. Перейди на code.visualstudio.com → натисни **Download**
2. Відкрий завантажений файл → встанови як звичайну програму
3. Запусти VS Code

---

## Крок 2 — Встанови Claude Code розширення у VS Code

1. У VS Code натисни іконку **Extensions** на лівій панелі (або `Cmd+Shift+X` на Mac / `Ctrl+Shift+X` на Windows)
2. У пошуку введи: `Claude`
3. Знайди **Claude Code** від Anthropic → натисни **Install**
4. Після встановлення з'явиться іконка Claude на лівій панелі

---

## Крок 3 — Увійди в акаунт

1. Натисни іконку Claude на лівій панелі
2. Натисни **Sign In** → відкриється браузер
3. Увійди в свій claude.ai акаунт (або створи — безплатно)
4. Поверни VS Code — ти залогований

> Потрібен план **Claude Pro** ($20/міс) або **Max** для повноцінної роботи.

---

## Крок 4 — Відкрий цей проект і запусти онбординг

1. Скачай ZIP і розпакуй у зручну папку
2. У VS Code: **File → Open Folder** → вибери розпаковану папку
3. Натисни іконку Claude на лівій панелі
4. Напиши: `/onboarding` → натисни Enter

Все інше Claude зробить сам.
