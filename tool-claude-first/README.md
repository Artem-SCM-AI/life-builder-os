# Claude Code Перше знайомство

Персональний Claude Code workspace — готовий за 15 хвилин.

---

## Крок 1 — Встанови Claude Code

1. Перейди на claude.ai/code → натисни **Download**
2. Відкрий завантажений файл → встанови як звичайну програму
3. Запусти Claude Code
4. Натисни **Sign In** → увійди у свій claude.ai акаунт (або створи — безплатно)

> Потрібен план **Claude Pro** ($20/міс) або **Max** для повноцінної роботи.

---

## Крок 2 — Відкрий цей проект

1. Скачай ZIP і розпакуй у зручну папку
2. У Claude Code: натисни **Open Folder** → вибери розпаковану папку

---

## Крок 3 — Запусти онбординг

1. Напиши: `/onboarding` → натисни Enter

Все інше Claude зробить сам.
