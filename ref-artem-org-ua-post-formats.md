# @artem.org.ua — Post Formats
*Ukrainian-language personal brand. Last updated: June 2026.*

---

## #1 Формат: Хронологічний розділ життя

**Принцип:** Реальна дата → конкретна подія → рішення → наслідок. Без коментарів і узагальнень.

**Структура:**
```
[Вік або рік]: [факт].
[Вік або рік]: [факт].
[Різкий поворот — одним рядком].
[Що сталося далі — конкретно].
```

**Приклад з origin story:**
```
В 15 створив школу брейкдансу.
В 16 — діджей в нічному клубі.
В 17 закінчив школу і сів на наркотики.
9 років. Барна стійка. Весь СНД.
```

**Чому працює:** "Ефект серіалу" — читач хоче знати що далі. Автоматично виникає питання "і що потім?"

## #2 Формат: Рішення-пост

**Принцип:** Конкретна ситуація → рішення → результат. Без аналізу чому це правильно.

**Структура:**
```
[Ситуація одним реченням].
Я прийняв рішення.
[Яке рішення — конкретно].
[Що з цього вийшло — конкретно, з числом].
```

**Приклад:**
```
Кінець 2022. Банкрутство. Немає стратегії.
Я прийняв рішення.
Повернутись в найм. Взяти весь свій досвід і обрати новий напрямок.
Менш ніж за 5 років — Head of SC в компанії з $25M+ реєстру.
```

## #3 Формат: Число-пост

**Принцип:** Одне конкретне число пояснюється без бекграунду.

**Структура:**
```
[Велике число або несподіване співвідношення].
[Одне речення — звідки].
[Що це означає на практиці — без висновків].
```

**Приклад:**
```
$91,800.
Стільки один клієнт переплатив 3PL за 8 місяців.
Не через обман. Через відсутність аудиту рахунків.
```

## #4 Формат: Розвінчування міфу

**Принцип:** Популярна заява → "Неправда" або "Це не так працює" → реальний факт з особистого досвіду.

**Структура:**
```
"[Популярна думка]."
Я так думав 3 роки. Потім помилився.
[Конкретна помилка — що трапилось].
[Реальна механіка — одне речення].
```

---

## #5 Формат: Покроковий гайд

**Принцип:** Конкретна задача → конкретні кроки → конкретний результат. Починати відразу з кроку 1, без вступу.

**Структура:**
```
Як я [конкретна задача] за [час]:

Крок 1: [дія].
Крок 2: [дія].
Крок 3: [дія].

Результат: [що отримав, числа].
Збережи — знадобиться.
```

## #6 Формат: Бар'єр → реальність

**Принцип:** Назвати головний страх аудиторії прямо → не продавати рішення → просто сказати що насправді правда.

**Структура:**
```
Ти думаєш [страх — конкретно].
Це реальна думка. Я теж так думав.
[Що насправді — одне-два речення, без рішення і CTA].
```

**Чому без CTA:** Довіра будується на тому що ти озвучив страх і не спробував одразу продати. Продаж приходить через збереження і повернення до посту.

## #7 Формат: Чеклист

**Принцип:** 10+ пунктів конкретних ознак, помилок або тригерів. Без буллетів — тільки нумерація або перерви рядків.

**Використання:** Коли хочеш зберігань. Чеклисти зберігають найчастіше.

**Структура:**
```
[Кількість] ознак що [ситуація]:

1. [Ознака — конкретна].
2. [Ознака — конкретна].
...
10. [Ознака].

Якщо 3+ — час щось міняти.
```

---

## #8 Формат: Провальний кейс з числом

**Принцип:** Конкретна помилка → конкретна втрата → що зрозумів (без узагальнень).

**Структура:**
```
Я втратив [число] через [одне речення — причина].

[2-3 речення що сталося — без жалості до себе].

Зараз роблю по-іншому: [одна конкретна зміна].
```

## Технічні правила форматування

**Довжина поста в треді:** 3–12 рядків. Кожен пост — одна стиснута думка.

**Рядки:** Один рядок = одне речення. Якщо речення варте окремого рядка — ставити Enter.

**Фрагменти — нормально:**
"Три роки ріелтором у Дніпрі. Я був поганим ріелтором." Обидва речення коректні.

**Списки без буллетів:**
```
В 15: школа брейкдансу.
В 16: діджей.
В 17: наркотики.
```
Не:
```
• В 15 я відкрив школу брейкдансу
• В 16 почав диджеювати
• В 17 потрапив у залежність
```

**Відео:** Тільки якщо є конкретна демонстрація (інструмент, результат). Не для "поговорити". Текст — основний формат.
