# @artem.org.ua — CTA Strategy
*Ukrainian-language personal brand. Last updated: June 2026.*

---

## Загальний принцип

**Ніколи не продавати в тілі поста.** Пост будує довіру. CTA — легкий наступний крок. Жорсткий продаж в тексті вбиває і те, і інше.

**Ціннісна ланцюжок:**
Збереження → Повернення → Довіра → Переходи → Продажі

Найкраще продають пости які взагалі не продають — а дають настільки цінний або зворушливий контент, що читач сам знаходить посилання в біо.

## Типи CTA по меті

### 1. CTA на підписку
**Коли:** Пост з особистою történетью або сильним нішевим інсайтом.
**Формат — пряме запрошення без тиску:**
```
Будую в публічному режимі. Підписуйся якщо цікаво.
```
```
Наступний пост — про [тему]. Щоб не пропустити — підписатись.
```
**Не використовувати:** "Підписуйтесь, якщо хочете успіху!" — звучить відчайдушно.

### 2. CTA на збереження
**Коли:** Чеклист, покроковий гайд, число-пост.
**Формат:**
```
Збережи — повернешся.
```
```
Знадобиться — збережи зараз.
```
**Тільки один раз. В кінці поста. Не посередині.**

### 3. CTA на коментар
**Коли:** Питання-пост або розвінчування міфу.
**Формат — конкретне питання, не відкрите:**
```
А ти через таке проходив?
```
```
Скільки годин на тиждень витрачаєш на це вручну?
```
**Не:** "Що думаєте?" (занадто відкрито, ніхто не відповідає)

### 4. CTA на продукт
**Коли:** Пост прямо пов'язаний з продуктом — кейс, результат, демо.
**Формат — м'яке входження:**
```
Якщо хочеш спробувати — посилання в біо.
```
```
Зробив з цього інструмент. Деталі в біо.
```
**Правило:** Не більше одного посилання в пості. Один CTA на один пост.

### 5. Закріплений коментар (pinned reply)
**Коли:** Пост набирає трекшн.
**Дія:** Закріпити власний коментар з посиланням на продукт або наступний пост серії.
**Формат:**
```
Якщо цей пост резонує — [посилання на продукт або наступний крок].
```

## Що НЕ писати

- "DM мені слово X" — відчуває себе маніпуляцією
- "Тільки сьогодні" / "Обмежена пропозиція" — для цього акаунту не підходить
- Ціна в тілі поста (тільки в біо або в прямій відповіді на запит)
- Два і більше CTA в одному пості
- CTA в першому абзаці
